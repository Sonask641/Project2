
# E-commerce Automation Testing Project (Demoblaze)

## Project Title
Automated Testing of Web Application - https://www.demoblaze.com/

---

## Objective
To automate testing of an e-commerce website by simulating user actions and validating key functionalities like login, signup, product browsing, cart, order placement, navigation, UI, and responsiveness.

---

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Eclipse / IntelliJ
- Chrome Browser

---

## Features Tested

### Login
- Valid login
- Invalid login
- Empty fields
- Password masking

### Signup
- New user signup
- Existing user signup
- Empty fields validation

### Cart
- Add product to cart
- Add multiple products
- Remove product from cart
- View cart

### Order
- Place order successfully
- Order confirmation popup

### Product Browsing
- Phones, Laptops, Monitors categories
- Product details
- Navigation links

### Logout
- Logout functionality

### UI & Responsive
- Button visibility
- Layout check
- Mobile and tablet view

---

## Test Execution
- Framework: TestNG
- Run using: testng.xml
- All tests executed automatically
